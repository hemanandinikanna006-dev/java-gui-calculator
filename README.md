# Version 1
It is a simple calculator that performs arithmetic operations on two numbers.
Project Overview:
This calculator is built using Java Swing and currently supports basic arithmetic operations between two numbers. This is Version 1, and I plan to enhance it in future versions to support multiple operations and continuous calculations.

## Features:
1.Developed using Java Swing.
2.Supports the operations:
   1.Addition(+)
   2.Subtraction(-)
   3.Multiplication(*)
   4.Division(/)
3.Handles division by 0 using Exception Handling.
4.Displays integer without decimal when the result is in integer.
5.Simple and User-Friendly GUI Design.

## Concepts Used:
 1.Java Swing (GUI Components)
 2.Event Handling (ActionListener)
 3.Exception Handling
 4.Conditional Formatting for Output
 
 # Version 2
 The Calculator is improved with advanced functionality.
 
 ## Features:
 1.Multiple operations can be handled in a single expression.
 2.Operator precedence is handled.
 3.Expression is evaluated using a stack.
 4.Additional operations added:
    1.square root(√)
    2.Modulus(%)
    3.Power(^)
5.Operation history is displayed using a text area.
6.Styled calculator using UI.
7.Multhi-threading - Added a scrolling title header.

## Technologies Used

- Java Swing
- Stack Data Structure
